# **Краткое содержание**
Учебный проект для освоения базовых принципов интерактивного нарратива через работу с готовой диалоговой системой (визуальная новелла / диалоговая игра).

## Цель и задачи тренажера

Дать первый практический опыт работы с диалоговой системой в Unity — от запуска готовой визуальной новеллы до создания собственных ветвящихся диалогов и простых условных сценариев.

**Основные задачи студента:**

- Скачать/клонировать проект с GitHub (`gd-trainer-02-dialogue`)
- Открыть проект через Unity Hub и запустить сцену `DemoScene`
- Разобраться в устройстве UI для диалогов (панели, текст, кнопки выбора)
- Понять структуру ScriptableObjects для диалогов и условий
- Настроить внешний вид диалогового UI через Inspector
- Настроить переходы между экранами (меню, настройки, игра, пауза)
- Создать собственный ветвящийся диалог на основе шаблона
- Добавить простые условия показа диалогов на основе состояния игрока
- Протестировать и отладить нарративную систему по чек-листу

## **Требование к ПО**
* Unity Hub
* Unity Editor 6000.3.13f1 LTS
* Git

## Инструкция по интеграции проекта из GitHub в UnityHub

1. Склонируйте репозиторий gd-trainer-02-dialogue
   через GitHub Desktop или Git:
   • Откройте ссылку на репозиторий из материалов курса.
   • Нажмите кнопку Code → выберите Open with GitHub Desktop или скопируйте URL для git clone .
   • Укажите папку на диске для проектов курса.
   • Дождитесь окончания клонирования.
2. Добавьте проект в Unity Hub:
   • Откройте Unity Hub → вкладка Projects.
   • Нажмите Add → Add project from disk.
   • Выберите папку gd-trainer-02-dialogue на диске.
   • Нажмите Select Folder.
3. Откройте проект в нужной версии Unity:
   • В списке проектов найдите gd-trainer-02-dialogue.
   • Убедитесь, что выбрана версия Unity 6.3 LTS.
   • Откройте проект и дождитесь импорта.
   • Проверьте вкладку Console.
4. Откройте сцену DemoScene:
   • В окне Project разверните Assets/_Project/Scenes .
   • Дважды кликните DemoScene.unity .
   • Убедитесь, что в Hierarchy отображаются основные объекты (Canvas, NPC, менеджеры).
5. Запустите диалоговую сцену:
   • Нажмите Play.
   • Переключитесь на вкладку Game.
   • Пройдите диалог до конца, прочитав все реплики.
   • Обратите внимание, как меняются варианты ответов и реплики.
6. Найдите ScriptableObjects диалогов:
   • В окне Project разверните Assets/_Project/Dialogues .
   • Просмотрите несколько DialogueNode(двойной клик по asset).
   • Найдите поля speakerName, dialogueText, choices.
7. Найдите Canvas диалога:
   • В Hierarchy найдите объект DialogueCanvas.
   • Выделите его и посмотрите структуру UI: DialoguePanel, SpeakerName, DialogueText, контейнер кнопок.
   • Убедитесь, что элементы совпадают с тем, что вы видели в Game View.
