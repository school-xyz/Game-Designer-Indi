Краткое содержание
-
- Учебный проект для освоения Unity с нуля через работу с готовой гиперказуальной игрой в жанре endless runner/collector.

Требование к ПО
-
- Unity Hub
- Unity Editor 6000.3.13f1 LTS
- Git

Инструкция по интеграции проекта из GitHub в UnityHub
-
Склонируйте репозиторий gd-trainer-01-hypercasual через GitHub Desktop:
- Откройте ссылку на репозиторий из материалов курса.
- Нажмите кнопку “Code” → выберите Open with GitHub Desktop
- Если GitHub Desktop ещё не установлен, установите его по подсказке GitHub и повторите шаг.
- В окне GitHub Desktop выберите папку на диске, куда хотите сохранить проект (можете создать отдельную папку для учебных проектов).
- Нажмите Clone и дождитесь, пока полоска прогресса дойдёт до конца.

Добавьте проект в Unity Hub:
- Откройте Unity Hub и перейдите на вкладку “Projects”
- Нажмите кнопку “Add” (или значок “+”) → Add project from disk.
- В проводнике выберите папку, в которую вы только что клонировали gd-trainer-01-hypercasual (корневая папка проекта).
- Нажмите Select Folder.

Откройте проект в нужной версии Unity и дождитесь первого импорта:
- В списке проектов Unity Hub найдите gd-trainer-01-hypercasual.
- Убедитесь, что для проекта выбрана версия Unity, указанная в курсе (Unity 6.3 LTS). Если нет — выберите её в выпадающем списке.
- Нажмите по проекту, чтобы открыть его в Unity.
- При первом запуске дождитесь окончания импорта (обычно 2–5 минут).
- Откройте вкладку “Console” и проверьте, что нет красных ошибок (“Error”) и критических предупреждений.

Откройте игровую сцену GameScene:
- В окне “Project” разверните папку Assets/_Project/Scenes
- Найдите файл сцены GameScene.unity и дважды кликните по нему
- Убедитесь, что в окне “Hierarchy” отображаются основные объекты сцены (игрок, монеты, препятствия, GameManager, UI и т.д.)

Запустите игру и поиграйте не меньше 1 минуты:
- Нажмите кнопку “Play” в верхней части Unity
- Перейдите во вкладку “Game”, чтобы видеть игру глазами игрока
- Поиграйте ~1 минуту: управляйте персонажем (клик/тап/клавиша — как задано в проекте), собирайте монеты, постарайтесь хотя бы один раз столкнуться с препятствием, чтобы увидеть Game Over.
- Наблюдайте, как меняется счёт при сборе монет и что происходит при проигрыше
- Нажмите Play ещё раз, чтобы остановить игру
